  Trick đơn giản nhất: kiểm tra con trỏ trước khi dùng.

  // Nguy hiểm: dereference thẳng không kiểm tra
  int* ptr = getPointer();
  cout << *ptr; // crash ngay nếu ptr == nullptr

  // An toàn: kiểm tra nullptr trước khi dùng
  int* ptr = getPointer();
  if (ptr != nullptr) {
      cout << *ptr;
  } else {
      cout << "Con trỏ rỗng, cần xử lý ở đây";
  }

  Từ C++11 trở đi, luôn dùng nullptr thay vì NULL hay số 0 — compiler hiểu rõ
  hơn và có thể bắt lỗi ngay lúc biên dịch thay vì để đến lúc chạy mới crash.
