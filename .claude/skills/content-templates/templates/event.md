# Template: Event Post (Fanpage trung tâm lập trình)

## Cấu trúc 4 phần — tổng 150-220 từ

Event post khác announcement: tập trung vào **trải nghiệm** và **giá trị nhận được**, không chỉ thông báo.

### 1. Hook — trải nghiệm hứa hẹn (1-2 câu, ~20 từ)

Mở bằng điều thú vị sẽ xảy ra tại event, không phải tên event.

Ví dụ:
- "Thứ 7 này bạn sẽ viết API đầu tiên trong 2 tiếng — dù chưa biết backend là gì."
- "Buổi workshop cuối tháng sẽ có một senior dev chia sẻ những sai lầm đã khiến bạn ấy mất 6 tháng đầu đi làm."

### 2. Thông tin event (3-4 dòng, ~50 từ)

Ngắn gọn, dùng icon:
```
📅 [Ngày] | [Giờ bắt đầu - kết thúc]
📍 [Địa điểm / Online link]
👥 Số lượng: [giới hạn slot nếu có]
🎟 Vé: [Miễn phí / Giá / Link đăng ký]
```

### 3. Agenda / Nội dung sẽ diễn ra (4-6 câu, ~80 từ)

Nêu cụ thể người tham dự sẽ làm/học/nhận được gì. Tránh mô tả mơ hồ.

Format gợi ý:
- Dùng danh sách ngắn (3-4 bullet) với emoji ✅ hoặc 👉
- Mỗi bullet = 1 kết quả cụ thể hoặc hoạt động

Ví dụ:
```
✅ Hiểu REST API là gì và khi nào cần dùng
✅ Viết endpoint CRUD đầu tiên bằng Node.js + Express
✅ Test API với Postman (có hướng dẫn từng bước)
✅ Q&A trực tiếp với instructor
```

### 4. CTA (1-2 câu, ~20 từ)

Urgency thật (slot giới hạn, deadline đăng ký) — không tạo urgency giả.

Ví dụ:
- "Còn [X] slot — đăng ký tại link bio hoặc nhắn tin mình. Bạn quan tâm lĩnh vực nào nhất trong event này?"
- "Free hoàn toàn nhưng cần đăng ký trước để chuẩn bị tài liệu. Bạn sẽ tham gia chứ?"
